Four endpoints, one prepaid balance

Create a watch

$0.10 · x402 · seeds $0.10 credit

POST /v1/private/watch
Body: {chain, address, viewKey, webhookUrl}. We poll every 3 min, POST a signed event on every balance change.

Top up credit

$0.10 / $1.00 / $5.00 · x402 tiers

POST /v1/private/topup{,-1,-5}
$1 buys ~50 idle days; $5 buys ~250. The low_credit webhook tells you when to top up.

Historical lookup

$0.50 · one-off · view key in-memory only

POST /v1/private/historical
Returns received / spent / spendable atomic totals and optional per-note breakdown. Auto-detects birthday height for Zcash.

FREE · 6/min/IP · Zcash UFVK from a phrase

How the credit meter works

Try it now — derive a Zcash viewing key from a BIP-39 phrase (free, in-browser)

Calls POST /v1/private/derive-viewkey on api.seneschal.space. Returns a read-only Unified Full Viewing Key (UFVK) you can then plug into /v1/private/watch or /v1/private/historical.

Security warning. Your seed phrase will be sent over TLS to our server. We do not log or persist it, but a network attacker between you and us would see the bytes. For maximum safety, derive offline using the orchard-scanner binary on a trusted machine and skip this convenience endpoint. Use a throw-away test seed if you just want to see it work — the canonical BIP-39 vector abandon abandon … art is fine.

Other premium endpoints (x402 micropayments)

Pay-per-call endpoints on Base mainnet, settled in USDC via the x402 protocol — no API keys, no subscriptions, no gas for the payer (the openx402.ai facilitator pays the EIP-3009 transfer gas):

  • GET /v1/premium/opportunities — $0.05 — uncapped at-risk catalogue joined with realised 7-day market intel, pre-ranked by expected value.
  • GET /v1/premium/builder-stats — $0.10 — per-builder bid distribution (p25/median/p90/p99/max ETH) and hourly slot histogram from our shadow recorder.
  • GET /v1/q/* — $0.001 — Penny Oracle. Fifteen atomic single-fact endpoints across DeFi (liquidatable? at-risk count? top builder?) and privacy-chain (Monero/Zcash chain-tip facts from our nodes).
